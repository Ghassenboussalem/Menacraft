AI Pipeline
Blue + Red + Synthesis
The extension runs entirely in the browser — reading the already-rendered DOM without triggering Instagram's anti-bot protections (TLS fingerprinting, IP bans). No Instagram credentials are stored.
Agent Pipeline
Inspired by cybersecurity red/blue team methodology. Both teams independently analyse the same post, then a Synthesis Agent reconciles their findings into a single verdict.
Blue Team — Detection
Image Forensics
EfficientNet B4 (DFDC fine-tune)
GAN artifacts, splicing, copy-move detection
OCR + Claim Checker
Tesseract v5 → Gemini 1.5 Flash
Text overlaid on image vs. caption claim
Link Scanner
Google Safe Browsing API v4
Phishing domains, spoofed news URLs
Metadata Analyser
ExifTool + heuristics
Stripped/forged EXIF, GPS anomalies
Red Team — Verification
Reverse Image Search
Google Vision API / CLIP
Historical reuse; different event or country
Caption–Image Alignment
OpenAI CLIP / SigLIP
Semantic mismatch between text and visual
Bot Pattern Detector
Comment frequency heuristics
Identical comments, no-profile accounts
Source Credibility
Gemini 1.5 Flash + domain DB
Writing style, account age, domain trust
Synthesis Agent
Receives JSON outputs from both teams. Weights findings by category severity. Produces final Verdict + Confidence Score (0–100) + 2-sentence plain-language explanation covering all 3 functional axes.
